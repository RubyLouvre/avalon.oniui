accordion

按钮组件 按钮组件提供丰富的样式、形式选择，除与bootstrap可用的button样式保持一致外，支持small、default、big、large四种尺寸，同时支持图标button，可以是仅有图标的button，图标在左边的button、图标在右边的button、两边都有图标的button，当然也支持图标组，有水平图标组、垂直图标组两种形式
使用说明
名字 默认值 说明
配置参数
groups 默认fakse, type: Boolean 是否是button组
direction 默认"", type: String button组的方向，有水平button组和垂直button组，默认是水平，可以设置为"vertical"
groupCorner 默认true, type: Boolean
data 默认[], type: "Array"

data属性配置button组的内容，每一个数组元素都是一个包含单个按钮基本信息的对象。

注意，请只在button组由至少两个按钮组成时，才配置button组件为button组，也就是设置groups为true时，且配置相应的data

当然还有一种直接列出button组内容的方式，不过这种情况需要制定组件名为buttonset，请看demo 4a>

data: [{
    type: "labeledIcon",
    iconPosition: "right",
    icon: "\&\#xf04c;",
    size: "large",
    color: "success",
    text: "暂停"
}, {
    type: "labeledIcon",
    iconPosition: "right",
    icon: "\&\#xf04b;",
    size: "large",
    color: "success",
    text: "播放"
}, {
    type: "labeledIcon",
    iconPosition: "right",
    icon: "\&\#xf074;",
    size: "large",
    color: "success",
    text: "拖曳"
}]
type 默认"text",type: String 配置button的展示形式，仅文字展示，还是仅图标展示，或者文字加图标的展示方式，三种方式分别对应："text"、"icon"、"labeledIcon"
iconPosition 默认"left",type: String 当type为icon或者labeledIcon时，定义icon在哪边，默认在text的左边，也可以配置为右边("right"),或者两边都有("left-right")
icon 默认"",type: String 当type为icon或者labeledIcon时，定义展示icon的内容，本组件的icon是使用web font实现，当iconPosition为"left"或者"right"时，将icon的码赋给icon，当iconPosition为"left-right",将left icon与right icon的码以"-"分隔，比如data-button-icon="\&\#xf001;-\&\#xf06b;"
size 默认"", type: String button有四个尺寸"small", "default", "big", "large"
width 默认："", type: Number 可以通过width定义button的宽度，必须是Number
color 默认""， type: String 定义button的颜色，默认提供了"primary", "warning", "danger", "success", "info", "inverse", "default" 7中颜色，与bootstrap保持一致
disabled 默认false, type: Boolean 配置button的禁用状态
corner 默认true， type: Boolean || Number 设置是否显示圆角，可以布尔值或者Number类型，布尔只是简单的说明显示或者不显示，Number则在表示显示与否的同时，也是在指定圆角的大小，圆角默认是2px。
label 默认"", type: String 设置button的显示文字，label的优先级高于元素的innerHTML

  2. 设置button的width和color
  3. 通过ms-widget="button, $, buttonConfig"的方式设置button组
  4. 通过ms-widget="buttonset"的方式设置button
