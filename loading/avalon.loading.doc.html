loading

loading组件

loading组件，实现各种加载动画效果

使用说明
名字默认值说明
配置参数
color "#619FE8" 效果的颜色
width 32 loading动画的宽度，圆形排列的外直径
height 32 loading动画的高度，如果不设置，默认等于width
widthInner 28 loading动画是圆形排列的时候，这个参数指的是内直径
type "ball" 类型，默认是ball，球，可取spin,ticks
toggle true 是否显示
modal true 是否显示遮罩
modalMpacity 0.1 遮罩透明度
container void 0 loading效果显示的容器，默认是绑定widget的元素
count 10 type=ball，loading效果组成的小图形个数
interval 120 type=ball，毫秒数，动画效果帧间隔
circleMargin 1 type=ticks，小球之间的间距，单位是一倍小球半径
count 3 type=ticks，小球个数
height 20 type=ticks，高度
interval 360 type=ticks，毫秒数，动画效果帧间隔
angel 90 type=spin，转动的弧形的角度，单位是1度
opacity 0.2 type=spin，背景圆弧的透明度
startangle 0 type=spin，圆弧开始的角度，单位1度
interval 36 type=spin，毫秒数，动画效果帧间隔
count 8 type=spokes，长方形个数
width 32 type=spokes，效果宽度,
spokesWidth 4 type=spokes，小长方形宽度
spokesHeight 8 type=spokes，小长方形高度
interval 125 type=spokes，效果动画间隔毫秒数
width 64 type=spinning-bubbles 宽度，小球的个数继承自type=ball
widthInner 54 type=spinning-bubbles 内宽
height 30 type=bubbles 高度，宽度继承type=spinning-bubbles
widthInner50 type=bubbles 内宽
count 3 type=bubbles 球的个数
interval 360 type=bubbles 动画ms数
circleMargin 0.5 type=bubbles bubbles效果下个小球的间距
onInit回调函数onInit(vmodel, options, vmodels) 完成初始化之后的回调,call as element's method
getTemplate配置接口getTemplate(tpl, opts, tplName) 定制修改模板接口
showLoading接口调用showLoading() 显示loading效果
hideLoading接口调用hideLoading() 隐藏loading
destroyLoading接口调用destroyLoading() 销毁loading

例子：

  1. loading:ball
  2. loading:spin
  3. loading:ticks
  4. loading:spokes
  5. loading:spinning-bubbles
  6. loading:bubbles-这个只有在高级非IE浏览器上才能work good
