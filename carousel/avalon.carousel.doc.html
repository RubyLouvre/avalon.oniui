carousel

carousel组件

图片轮播

使用说明
名字默认值说明
配置参数
pictures [] pictures 轮播图片素材
pictureWidth 500 pictureWidth 图片显示宽度
pictureHeight 200 pictureHeight 图片显示高度
effect "slide" effect 图片切换类型，默认为"slide"，取值：none:无特效 / fade:渐隐 / slide:滑动
easing "easeInOut" effect 缓动类型，默认为"easeInOut"，取值 linear:无缓动效果 / easeIn:在过渡的开始提供缓动效果 / easeOut:在过渡的结尾提供缓动效果 / easeInOut 在过渡的开始和结尾提供缓动效果
timeout 2500 timeout 切换时间间隔
during 300 during 切换速度，越小越快，单位为毫秒
alwaysShowArrow true alwaysShowArrow 显示左右切换箭头
alwaysShowSelection true alwaysShowSelection 显示底部圆形切换部件
autoSlide true autoSlide 自动播放
hoverStop false autoSlide 鼠标经过停止播放
adaptiveWidth false adaptiveWidth 适应外围宽度，为true时指定pictureWidth不起作用
adaptiveHeight false adaptiveHeight 适应外围高度，为true时指定pictureHeight不起作用
arrowLeftNormalSrc "./images/arrows-left-icon.png" arrowLeftNormalSrc 左箭头正常状态图标
arrowRightNormalSrc "./images/arrows-right-icon.png" arrowLeftNormalSrc 右箭头正常状态图标
arrowLeftHoverSrc "./images/arrows-left-hover-icon.png" arrowLeftNormalSrc 左箭头hover状态图标
arrowRightHoverSrc "./images/arrows-right-hover-icon.png" arrowLeftNormalSrc 右箭头hover状态图标
onInit回调函数onInit(vmodel, options, vmodels) 完成初始化之后的回调,call as element's method
getTemplate配置接口getTemplate(tpl, opts, tplName) 定制修改模板接口
setArrowVisible接口调用setArrowVisible() hover在组件上时使Arrow显示，停止轮播
setArrowHidden接口调用setArrowHidden() hover离开时使Arrow隐藏，重新开始轮播
animate接口调用animate(direct, distance) 图片滚动，direct为方向（1/-1），distance为距离（>0整数）
getOpacity接口调用getOpacity(index) fade effect 下改变前后图片透明度
selectPic接口调用selectPic(index) 通过底部圆形选择图片
arrowHover接口调用arrowHover(direction) 左右箭头hover事件
arrowBlur接口调用arrowBlur(direction) 左右箭头blur事件
autoPlay接口调用autoPlay(vmodel) 自动开始轮播
