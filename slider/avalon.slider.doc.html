slider

slider组件用来拖动手柄选择数值，可以水平拖动、垂直拖动、设置range使得两边都可以拖动，或者根据设置的步长更新滑块数值

使用说明
名字 默认值 说明
配置参数
max 100 组件的最大值
min 0 组件的最小值
orientation "horizontal" 组件是水平拖动还是垂直拖动，垂直是“vertical”
range false 滑块是否显示滑动范围，配置值可以是true、min、max

true: 显示滑动范围

min: 滑块值最小的一端固定

max: 滑块值最大的一端固定
step 1 滑块滑动的步值
value 0 滑块的当前值，当range为true时，value是滑块范围表示的两个值，以“,”分隔
disabled false 是否禁用滑块, 设为true时滑块禁用
接口方法与固有属性
values null 当range为true时，values数组需要有两个值，表示滑块范围
onDragStart(event, data) 滑动开始的回调，参数是事件对象event和保存滑动信息的data
onDrag(vmodel, data) 滑动时的回调,参数是事件对象event和保存滑动信息的data
onDragEnd(event, data) 滑动结束时的回调，参数是事件对象event和保存滑动信息的data
