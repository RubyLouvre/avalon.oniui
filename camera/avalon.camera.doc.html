camera

camera组件

基于图片轮播组件，增减了更多动画效果。注意：为保证视觉效果，图片尺寸最好与组件尺寸相同。

使用说明
名字默认值说明
配置参数
pictures [] 轮播图片素材
effect "random" 图片切换类型，取值：random:全部随机 / none:无特效 / slideX:横向滑动 / slideY:纵向滑动 / fadeIn:渐入 / crossX:横向交叉 / crossY:纵向交叉 / stepX:横向阶梯 / stepY:纵向阶梯 / rotateFadeIn:旋转式渐入
timeout 2500 切换时间间隔
during 600 切换速度，越小越快，单位为毫秒
alwaysShowArrow true 显示左右切换箭头
alwaysShowSelection true 显示底部圆形切换部件
hoverStop true 鼠标经过停止播放
adaptiveWidth false 适应外围宽度
adaptiveHeight false 适应外围高度
eventType "click" 触发tab切换的nav上的事件类型，取值click\mouseenter\both
arrowLeftClass "" 左右箭头的className，可不传
arrowRightClass "" 左右箭头的className，可不传
slicedCols 5 区块列数
slicedRows 4 区块行数
onInit回调函数onInit(vmodel, options, vmodels) 完成初始化之后的回调,call as element's method
getTemplate配置接口getTemplate(tpl, opts, tplName) 定制修改模板接口
