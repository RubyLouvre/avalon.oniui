notice

notice

notice组件用来向用户显示一些提示信息

使用说明
名字 默认值 说明
配置参数
type "info" notice类型,可以选择为"success"、"error"或者默认的"info"
header/title "notice title" notice的标题
timer 默认为0 notice显示之后自动隐藏的定时器，0表示不自动隐藏
hasCloseBtn 默认true 是否显示关闭按钮，设为false不显示
toggle 默认false 显示或隐藏notice， true显示，false隐藏
isPlace true 是否占位，false不占位，true占位，占位时notice显示在自定义的container中，不占位时append到body元素下
isAffix false 是否吸顶，非占位元素不吸顶，占位元素当滚动距离大于元素距页面顶部距离时吸顶，否则保持原位置
successClass "ui-notice-info" type为success时的提示类名
errorClass "ui-notice-danger" type为success时的提示类名
infoClass "" type为info时的提示类名
接口方法与固有属性
container 默认为绑定组件的元素 显示notice的容器，可以配置为自己想要包裹notice的元素对象或者元素id，id的话必须是"#ID"的格式
content 默认为"notice content" 要显示的内容,可以是DOM对象|String|DOM String
onHide(data, vmodels) 关闭回调
onShow(data, vmodels) 显示回调

例子：

  1. notice组件概览，具体使用demo及说明请看avalon.notice.ex1...6

  3. 自定义notice的container，设置isPlace可以决定notice是占位还是不占位
  4. 当配置notice占位且可吸顶时，notice会在页面滚动到它所在位置时吸顶显示
  5. 自定义notice的回调
  6. 设置timer使得notice在显示一段时间之后隐藏
  7. 自定义notice type的样式类
