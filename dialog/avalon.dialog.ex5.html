模拟alert（type="alert"）

说明：通过将dialog配置对象的type属性设为alert，将模拟alert的表现，也就是说不会存在“关闭”、“取消”按钮

                <!DOCTYPE HTML>
                <html>
                <head>
                    <meta charset='utf-8'/>
                    <title>模拟alert</title>
                    <script src='avalon.js'></script>
                </head>
                <body>
                    <div class="demo-show" ms-controller="demo">
                        <fieldset>  
                            <legend>模拟alert</legend>
                                <button ms-click="show('bb')">打开对话框</button>
                                <div ms-widget="dialog,bb,opts">
                                    呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵
                                </div>
                        </fieldset>
                        <script> 
                            require(["dialog/avalon.dialog"], function(avalon) {
                                avalon.define("demo", function(vm) {
                                    vm.opts = {
                                        type: 'alert',
                                        title: '警告框demo'
                                    };
                                    vm.show = function( id ){
                                        avalon.vmodels[id].toggle = true;
                                    }
                                })
                                avalon.scan();
                            })
                        </script>
                    </div>
                </body>
                </html>
